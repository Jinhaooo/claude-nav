# Privacy Policy / 隐私政策

## English

Claude Nav respects your privacy.

- **No personal data is collected.** The extension does not gather, transmit, or log any user information, conversation content, or browsing activity.
- **Local-only storage.** All settings are stored exclusively in `chrome.storage.sync`, which is managed by your browser. No data is sent to any external server operated by the author.
- **No third-party sharing.** No data of any kind is shared with, sold to, or disclosed to any third party.
- **No analytics or tracking.** The extension contains no analytics SDK, telemetry, or tracking pixels.

The extension only reads the DOM of `claude.ai` pages locally in order to render the navigation sidebar. Nothing leaves your device.

If you have any questions, please open an issue on the project's GitHub repository.

---

## 中文

Claude Nav 尊重并保护你的隐私。

- **不收集任何个人数据。** 本扩展不会采集、上传或记录任何用户信息、对话内容或浏览行为。
- **仅本地存储。** 所有设置仅存储于浏览器自带的 `chrome.storage.sync` 中，作者不会将任何数据发送到外部服务器。
- **不与第三方共享。** 任何数据都不会被共享、出售或披露给任何第三方。
- **无统计与追踪。** 扩展内不包含任何统计 SDK、遥测或追踪代码。

扩展仅在你本地的 `claude.ai` 页面读取 DOM 以渲染导航侧栏，所有处理均在本地完成，不会离开你的设备。

如有疑问，欢迎在项目 GitHub 仓库提 issue。
